# Types of constants in C++

- Literal constants
- Declared constants
  - `const` keyword
- Constant expressions
  - `constexpr` keyword
- Enumerated constants
  - `enum` keyword
- Defined constants
  - `#define`

## Literal constants

- Character Literal Constants (escape codes)
- `\n`
  - newline
- `\r`
  - return
- `\t`
  - tab
- `\b`
  - backspace
- `\'`
  - single quote
- `\"`
  - double quote
- `\\`
  - backslash

## Defined constants

- Constants declared using the `const` keyword

```cpp
#define pi 3.1415926
```

but, Using defined constants is not recommended in Modern C++
